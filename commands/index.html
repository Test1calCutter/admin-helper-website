Command Description Usage
/help Will show all commands. /help
/ban Will ban selected user. /ban [user] [reason]
/kick Will kick selected user. /kick [user] [reason]
/mute Will mute selected user. /mute [user] [time] [reason]
/unmute Will unmute muted user. /unmute [user]
/info Two usages. Either to retrieve server info or to get user information. /info or /info [user]
/afk Will set afk status. /afk (reason)
/safety Will show a modal to set up enhanced safety. /safety
/blacklistadd Used to add words to the blacklist. Must have enhanced safety enabled. /blacklistadd [word]
/blacklistremove Used to remove words from the blacklist. /blacklistremove [word]
/blacklistimport Can be used to import a list of words using pastebin. /blacklistimport [link]
/blacklistexport Can be used to export/remove a list of words using pastebin. /blacklistexport [link]
/blacklist Will show all blacklisted words. /blacklist
/8ball The almighty Magic 8ball will answer your questions. /8ball [text]
/activity Will tell you what you can do if youre bored. /activity
/about Will tell you about Admin Helper. /about
/commandcreate Will create a custom command.
Color must be in hex		 /commandcreate [name] [title] [description] (url) (hex)
/deletecommand Will delete a user created command. /deletecommand [name]
/listcommands Will list all user created commands. /listcommands
/incidents Will list current incidents. /incidents [component]
/incidentslist Will list all incidents. Prior and current. /kick [user] (reason)
/scheduled Will list scheduled incidents. /scheduled [component]
/screenshot Will list make a screenshot of a website. /screenshot [url]
/status Will show the uptime percentage. /status [component]
/t Will run a user created command. /t [name]
